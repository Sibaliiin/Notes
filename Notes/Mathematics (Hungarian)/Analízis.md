# Deriválási szabályok
Pár alap deriválási szabály:

$\displaystyle (\sin(x))' = cos(x)$ 
$(\cos(x))' = -sin(x)$
$(e^x)' = e^x$
$(x^n)' = nx^{n-1}$
$(c)' = 0$
$(x)' = 1$


$(f(x) + g(x))' = f'(x) + g'(x)$
$(f(x) - g(x))' = f'(x) - g'(x)$

$(f\cdot g)' = f'g + fg'$
$\displaystyle \left (\frac{f}{g} \right)' = \frac{f'g - fg'}{g^{2}}$
